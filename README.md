## Introduction

This is my personal project by using React, Node.js, Express & MongoDB, I learn how to build a Full Stack MERN Application.

The app is called **Minidum** which is the ultimate platform for learning and sharing IT Knowledge. It a simple social media app that allows users to learn useful knowledge and create their own content.

## Feature

- Authentication
- View a Blog post
- Like a Blog Post
- Post pagination
- Recomment relevant blog
- Admin content management (CRUD).
- Format and style blog detail by using Markdown.
- Blog search feature
